##  Key Features

- **Dynamic Hex Grid Generation**  
  Enter any number of rows and columns to create a custom honeycomb grid.

- **Interactive Start and End Selection**  
  Click on hex cells to select a starting point and destination. The path is automatically highlighted.

- **Shortest Path Highlighting**  
  The program calculates the shortest path between two cells and visually marks it.

- **Bee Animation with Sound**  
  Watch the bee move along the path with smooth animation while buzzing!

- **Background Music Toggle**  
  Enjoy optional background music while interacting with the grid. Toggle music ON/OFF anytime.

- **Retry & Back Options**  
  Easily retry your selections or go back to input a new grid size.

- **Informative Popup**  
  A custom popup shows detailed path information:
  - Start and End cell numbers
  - Number of cells passed
  - List of all cells in the path

- **Professional Look & Feel**  
  Vibrant colors, animations, and sound make the project engaging and visually appealing.
